## ❌ 失败的两个测试用例

**位置**: `tests/unittest/UT-agent-codegraph-native/agent-codegraph-native.test.ts`

### 1. `agent-codegraph-native > integration > init + search 小 fixture`
### 2. `agent-codegraph-native > integration > CodeGraphExplore 返回源码片段`

**失败原因**:
```
Error: Failed to open SQLite via better-sqlite3: The module 
'/Users/cuiyunfeng/workspace/AxeCoder/electron/main/codegraph/dist/node_modules/better-sqlite3/build/Release/better_sqlite3.node'
was compiled against a different Node.js version using NODE_MODULE_VERSION 121. 
This version of Node.js requires NODE_MODULE_VERSION 115.
```

**根本原因**: better-sqlite3 原生模块版本不匹配
- 当前 Node 需要 MODULE_VERSION 115
- 编译的模块使用 MODULE_VERSION 121

**解决方案**: 
```bash
cd electron/main/codegraph/dist
npm rebuild better-sqlite3
# 或从根目录
npm run postinstall
```
