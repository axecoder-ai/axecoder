# 选型记录

## 2a 选型摘要

**一句话需求：** 在聊天输入区模式选择下拉的 Agent 行右侧增加 Auto Plan 开关，就地切换 `agentAutoPlan`，无需跳转设置页。

| 维度 | 提案 1 下拉内嵌 SwitchToggle | 提案 2 下拉 + 移除设置页 | 提案 3 仅指示器 |
|------|------------------------------|--------------------------|-----------------|
| 核心思路 | 复用 `agentAutoPlan` 配置，下拉内 SwitchToggle | 同左，并删除设置页开关 | 圆点指示 + 跳转设置 |
| 主要改动 | Dropdown + ChatPane + App | 另改 GeneralTab | 导航逻辑 |
| 优点 | 最小改动、符合截图 | 单一入口 | 布局最窄 |
| 缺点 | 两处入口 | 设置页少一项 | 不符合开关交互 |
| 工作量 | 小 | 中 | 小 |
| 适合场景 | 快速落地 | 强收敛入口 | 不推荐 |

**关键差异：**
- 提案 1 保留设置页开关，行为与下拉同步。
- 提案 2 移除设置页重复项。
- 提案 3 不能就地切换。

**推荐：提案 1 – 下拉内嵌 SwitchToggle**

## 2b 用户确认

- **选定提案：** 提案 1 – 下拉内嵌 SwitchToggle（复用设置项，最小改动）
- **调整说明：** 开关使用更小尺寸（compact），避免撑宽下拉
