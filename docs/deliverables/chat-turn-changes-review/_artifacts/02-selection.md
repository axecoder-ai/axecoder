# 选型记录

## 2a 选型摘要

**需求回顾：** 优化 Agent 聊天底部「本轮文件变更」栏样式；点击 Review 打开对应文件的 diff 视图。根因：无编辑器标签时 `EditorPane` 未挂载，`onReviewPatch` 静默失败。

| 维度 | 提案 1 内嵌+Workbench | 提案 2 最小修复 |
|------|------------------------|-----------------|
| 核心思路 | 变更栏内嵌 input-box；`useWorkbench.openDiffTab` 统一入口 | App.vue 内联写 openFiles；栏外微调 CSS |
| 改动范围 | workbench + App + ChatPane + 栏组件 | App + 栏组件 CSS |
| 优点 | 架构清晰、SCM 等可复用 | 改动最小、上线快 |
| 缺点 | 面稍大 | diff 逻辑分散 |
| 工作量 | 中 | 小 |
| 适合场景 | 长期维护 | 快速修通 |

**推荐：** 提案 1（根因修复更彻底）

**调整说明（用户）：** 尽量贴近 Cursor——变更栏与输入框一体、Review 为主按钮。

## 2b 用户选择

- **选定提案：** 提案 2 – 仅修 Review 逻辑 + 变更栏外观微调
- **调整说明：** cursor-like 样式（内嵌一体 + Review 主按钮）；在提案 2 最小代码路径上补充必要的一体化样式
