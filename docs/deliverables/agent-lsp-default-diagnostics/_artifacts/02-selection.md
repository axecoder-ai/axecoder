# 选型记录

## 2a 选型摘要

**一句话需求：** Agent 链默认接通 LSP，Write/Edit/ApplyPatch 后自动把 LSP 诊断喂给 Agent（编辑器 LSP 已有，Agent 未默认接通）。

| 维度 | 提案 1 | 提案 2 |
|------|--------|--------|
| 核心思路 | 默认开 + 主进程 LSP 同步 | 提案 1 + 工具回报自动附诊断 |
| 改动范围 | config、sync、tool-executor | 再加 post-edit 诊断、新配置、会话预热 |
| 优点 | 改动小 | 自动喂诊断、体验完整 |
| 缺点 | 须手动 ReadLints | 略增延迟与回报长度 |
| 工作量 | 小 | 中 |

**推荐：** 提案 2

## 2b 用户选择

- **选定提案：** 提案 2 – 默认开启 + 同步 + 编辑工具自动附带诊断摘要
- **调整说明：** 无额外调整
