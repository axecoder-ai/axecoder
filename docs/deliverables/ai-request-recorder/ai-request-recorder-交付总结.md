## 概述

类似性能仪表盘的 **AI 请求录制器**：开始录制后实时记录每次模型调用的提示词与返回，Agent 另记录工具调用/结果；可脱离独立窗；结束可保存 JSONL 或清空。

## 使用方式

1. TitleBar 点击 **红点** → 打开底部「录制」Tab  
2. **开始录制** → 发聊天或跑 Agent  
3. 点击条目查看请求/返回  
4. **结束录制** → **保存日志** 或 **清空**  
5. **脱离主窗** 大屏查看；**收回** 回底部 Tab  

日志路径：`~/Library/Application Support/AxeCoder/ai-traces/`（macOS）
