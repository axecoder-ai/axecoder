# Multi-Agent 模式工具更新修复总结

## 问题描述

在 `multi-agent` 聊天模式下，Agent 会话切换到该模式后，工具列表（`activeTools`）未能正确更新为只包含 Task 和 Agent 工具，导致运行时 Agent 可能访问到不应该暴露的工具。

## 根本原因

**文件**: `electron/main/agent/chat-mode.ts:58-62`

原代码：
```typescript
if (mode === 'multi-agent') {
  session.revealedToolNames.add('Task')
  session.revealedToolNames.add('Agent')
  session.activeTools = getSessionActiveTools(allTools, session.revealedToolNames)
}
```

问题：缺少 `return` 语句，导致函数继续执行后续的默认工具添加逻辑（Read/Write/Edit 等），最终 `activeTools` 包含了所有基础工具，而非仅限 Task/Agent。

## 修复方案

在 `multi-agent` 分支末尾添加 `return`，确保工具列表严格限定为 Task 和 Agent：

```typescript
if (mode === 'multi-agent') {
  session.revealedToolNames.add('Task')
  session.revealedToolNames.add('Agent')
  session.activeTools = getSessionActiveTools(allTools, session.revealedToolNames)
  return  // ← 新增
}
```
