# 选型记录

## 2a 选型摘要

**需求回顾：** 将 Agent SSE 执行进度从圆点列表改为 Claude Code CLI 风格的紧凑终端风（工具行、活动 shimmer、推理块），适配 IDE 深色主题。

| 维度 | 提案 1 终端风组件 | 提案 2 仅样式 |
|------|-------------------|---------------|
| 核心思路 | 新组件 + 结构化 step | CSS 改现有 ul |
| 改动范围 | agent-progress + 新 Vue + ChatPane | ChatPane CSS |
| 优点 | 对齐度高、可折叠 | 改动极小 |
| 缺点/风险 | 中等工作量 | label 解析脆弱 |
| 工作量 | 中 | 小 |
| 适合场景 | 长期维护、接近 CC | 快速试水 |

**关键差异：**
- 提案 1 可存 toolName/summary，工具行与 CC 一致。
- 提案 2 无法可靠折叠与树形子代理展示。
- 提案 1 需扩展单测；提案 2 几乎不动测试。

**推荐：提案 1 – 终端风独立组件 + 结构化进度步**

## 2b 用户选择

- **选定：** 提案 1
- **调整说明：** 无额外调整
