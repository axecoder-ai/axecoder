# AxeCoder 页面布局与按钮清单

> 调研日期：2026-06-29  
> 主布局入口：`src/App.vue`  
> 关联文档：[功能清单.md](./功能清单.md)

---

## 1. 窗口类型

| 窗口 | 说明 | 布局差异 |
|------|------|----------|
| **主窗口** | 默认 IDE 工作台 | 完整五区布局（见下） |
| **Companion 窗口** | 标题栏「双窗口」弹出，专放聊天/会话 | 仅 AI 侧栏全宽；标题栏有「交换左右布局」 |
| **Metrics 窗口** | AI Performance 独立窗口 | 仅 `AiMetricsPanel` 全屏 |
| **Trace 窗口** | AI Request Trace 独立窗口 | 仅 `AiTracePanel` 全屏 + Dock 回主窗 |

---

## 2. 主窗口整体布局

```
┌─────────────────────────────────────────────────────────────────────────────┐
│ TitleBar（标题栏）                                                            │
├──────────┬──────────────────────────────────────────────┬───────────────────┤
│          │                                              │  ChatPane         │
│ Primary  │  WelcomePage（无标签时）                        │  （聊天主区）      │
│ Sidebar  │  或 EditorPane（有打开文件时）                   ├─────────┬─────────┤
│          │                                              │         │ Agents  │
│ 文件树/   │                                              │         │ Panel   │
│ 搜索/Git │                                              │         │ 会话历史 │
├──────────┴──────────────────────────────────────────────┴─────────┴─────────┤
│ BottomPanel（终端 / 输出 / 问题 / Metrics / Trace）                             │
├─────────────────────────────────────────────────────────────────────────────┤
│ StatusBar（状态栏）                                                           │
└─────────────────────────────────────────────────────────────────────────────┘

浮层：SettingsPanel、SettingsModal、CommandPalette、QuickOpenPalette
```

**可拖拽分隔条**（非按钮，调整宽度/高度）：

| 分隔条 | 位置 | 文件 |
|--------|------|------|
| 主侧栏 ↔ 编辑区 | 左侧竖条 | `App.vue` `.primary-split-handle` |
| 编辑区 ↔ AI 面板 | 右侧竖条 | `App.vue` `.editor-ai-split-handle` |
| 聊天区 ↔ 会话历史 | 竖条 | `App.vue` `.agents-split-handle` |
| 主区 ↔ 底栏 | 横条 | `App.vue` `.bottom-split-handle` |
